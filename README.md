# AI Global Business Growth Advisor

An AI-powered business intelligence system that helps users:
- Choose safe business locations
- Predict future growth
- Analyze flood/earthquake risks
- Suggest suitable business types
- Find international demand
- Connect with dealers/importers
- Optimize export routes

## Project Structure
```
AI-Business-Advisor/
├── frontend/          # React.js dashboard
├── backend/           # Flask API server
├── ml_models/         # Machine learning models
├── datasets/          # Training data
├── database/          # Database schemas
└── requirements.txt   # Python dependencies
```

## Technology Stack
- **Frontend**: React.js, Tailwind CSS, Chart.js, Leaflet.js
- **Backend**: Flask, Python
- **ML Models**: Scikit-learn, TensorFlow, XGBoost
- **Database**: Firebase
- **APIs**: Google Maps, OpenWeather, USGS, UN Comtrade

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Set up Firebase configuration
3. Run backend: `python backend/app.py`
4. Run frontend: `cd frontend && npm start`

## Features
- User authentication
- Location risk analysis (flood/earthquake)
- Business recommendations
- Growth predictions
- International trade analysis
- Dashboard with charts and maps
- AI chatbot for business guidance
